# hot_and_taste
🔥 Hot & Tasty – Digital Menu Card for Local Shop

This project is a colorful and interactive digital menu card designed for a local food shop. The menu features a fluid background, attractive glowing borders, and a stylish “HOT AND TASTY” fire-themed header.
It includes two main sections — Non-Veg and Veg — with a page-flip animation that gives the feel of turning a real menu book.

Each food item (like Chicken Burger, Chicken Roll, Paneer Roll, Veg Burger, and Corn Cheese Pizza) is displayed with beautiful animated entry effects, arranged dynamically in creative frames.

The design aims to make the digital ordering experience more modern, engaging, and eye-catching, especially for small local shops and cafés.

✨ Features

Dynamic fluid background with glowing borders

Realistic page-flip animation between Veg and Non-Veg pages

Stylish “HOT AND TASTY” header with fire effect

Smooth item animations and random positioning

Perfect for use as a QR-code-based digital menu

💻 Tech Used

HTML

CSS (with gradient and animation effects)

JavaScript (for page flip and animations)
